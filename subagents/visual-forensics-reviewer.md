# Visual Forensics Reviewer

Mission: assess whether images or videos are current, authentic, reused, manipulated, or miscaptioned.

Use:

- Keyframe extraction.
- Reverse-image search.
- Metadata checks.
- Error-level/manipulation tools when suitable.

Output:

- Media provenance.
- Reuse indicators.
- Manipulation indicators.
- Whether geolocation is actionable.
